# Common Mail

## The Foragers  
COMP3013 Group Project

**Common Mail** is a medieval fantasy delivery game where the player works as a mail carrier for the Common Mail service. The player accepts delivery jobs, collects parcels, locates the correct NPC recipients, and completes deliveries to earn experience and reputation.

The game was developed as a small single-player prototype using Godot 4.6 stable.

## Game Overview

You are a newly recruited member of the Common Mail, an organisation responsible for delivering mail across the fantasy region of Everstamp.

The player must accept delivery quests, collect parcels, find the correct villagers, and deliver the right items. Successful deliveries reward the player with experience and reputation, while incorrect deliveries can reduce reputation and cause NPCs to react negatively.

The game focuses on a compact delivery loop, light RPG progression, NPC interaction, and a stylised medieval fantasy world.

---

## Core Gameplay Loop

1. Start in the Common Mail hub.
2. Accept delivery jobs from the task board.
3. Collect the required parcels.
4. Travel through the world to find the correct NPC recipients.
5. Deliver the correct parcel.
6. Gain experience and reputation.
7. Spend skill points in the skill tree.
8. Continue improving delivery efficiency.

---

## Features

- Single-player fantasy delivery gameplay.
- Quest board for accepting delivery jobs.
- Pick-up and delivery system for parcels.
- NPC interaction system.
- NPC behaviour including idle, wandering, and hostile reactions.
- Reputation system.
- Experience and skill point progression.
- Skill tree interface.
- Inventory/mailbag system.
- Low-poly medieval fantasy environment.
- Day/statistics summary showing delivery performance.
- Keyboard and mouse controls.

## How to Run the Project

### Playing the Build

1. Download the latest release from the GitHub Releases page.
2. Extract the downloaded `.zip` file.
3. Run the exported executable.

### Opening the Project in Godot

1. Download or clone this repository.
2. Open Godot 4.6 stable.
3. Select **Import**.
4. Choose the project folder.
5. Open the project.
6. Run the main scene.

## Project Status

This project was developed as a university group prototype for COMP3013. The current version demonstrates the core gameplay loop, including delivery quests, NPC interaction, inventory use, reputation, and skill progression.

Future improvements could include:

- More delivery quests.
- More varied NPC reactions.
- Expanded class system.
- Randomised delivery addresses.
- Improved world art and environmental polish.
- More detailed reputation consequences.
